## 功能介绍

该脚本在90直播网页面添加了一个操作面板，方便用户快速进行交互操作。面板显示简单的消息和计数器，用户点击按钮即可增加计数。它还支持将内容复制到剪贴板，提升使用效率。

## 适用网站

- 90直播网

## 使用方法

1. 安装脚本后打开90直播网首页。
2. 页面底部会出现一个操作面板。
3. 点击面板上的按钮即可增加计数显示。
4. 可通过面板快速复制内容到剪贴板。

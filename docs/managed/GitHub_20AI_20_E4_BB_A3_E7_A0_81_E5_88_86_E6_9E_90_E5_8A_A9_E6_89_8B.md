## 功能介绍

本脚本会在 GitHub 仓库页面添加一个 AI 代码分析按钮。用户点击后，可通过 zread.ai 或 deepwiki.com 对仓库代码进行智能解读和分析。让你更方便地理解开源项目的代码结构和功能。无需手动复制代码到其他平台。

## 使用方法

1. 1. 安装脚本后，访问任意 GitHub 仓库页面。
2. 2. 在页面上方会看到新增的“AI Analysis”按钮。
3. 3. 点击按钮，选择 zread.ai 或 deepwiki.com 进行代码分析。
